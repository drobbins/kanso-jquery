# jQuery

[jQuery](http://jquery.com/) as a [Kanso](http://kan.so) module.

### Install

Add `jquery` to your dependencies section in `kanso.json`.

```javascript
...
  "dependencies": {
    "jquery": null,
    ...
  }
```

Run `kanso install` to fetch the package.

### Usage

Just `$ = require('jquery');` and you're good to go.
